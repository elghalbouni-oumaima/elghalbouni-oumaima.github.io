## Project Overview

Built a complete data warehouse solution to transform raw data into actionable insights. Designed ETL pipelines, data models, and visual dashboards to support business analysis.

### Key Highlights

- Designed relational and star schemas for analytics-ready warehouse
- Implemented ETL pipelines using **Apache Airflow** to automate data ingestion and transformation
- Created interactive dashboards in **Power BI** to visualize KPIs and trends
- Executed complex SQL queries for reporting and decision support

### Tools & Technologies

**SQL • Data Warehousing • Apache Airflow • Power BI • ETL**

### Outcome

Demonstrates full data engineering workflow: data ingestion → storage → analysis → visualization. Ready for real-world business applications.

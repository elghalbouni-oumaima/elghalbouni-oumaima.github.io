## Project Overview

Optimized large-scale FHV trip datasets using NoSQL and big data technologies to improve query performance and analytics capabilities. Focused on processing massive datasets efficiently and extracting actionable insights.

### Key Highlights

- Processed and analyzed massive trip datasets using **Apache Spark**
- Stored and managed data efficiently using **Hadoop/HDFS** and MongoDB collections
- Optimized queries with indexes and aggregation pipelines for faster results
- Generated insights on trip patterns, volumes, and trends

### Tools & Technologies

**MongoDB • NoSQL • Hadoop/HDFS • Apache Spark • Query Optimization**

### Outcome

Highlights expertise in big data tools, NoSQL modeling, and performance optimization for real-world datasets.
